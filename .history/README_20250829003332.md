1. **Navigate to the project directory:**
   ```bash
   cd /Users/tejas/Documents/MineSweeperJavaFX
   ```



If you don't have Maven installed:

1. **Install Maven using Homebrew:**
   ```bash
   brew install maven
   ```

2. **Compile the project:**
   ```bash
   mvn clean compile
   ```

3. **Run the game:**
   ```bash
   mvn javafx:run
   ```
